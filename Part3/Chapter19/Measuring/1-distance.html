Measuring distances and finding points on a great arc

You can measure geographical distances in radians (and convert to km or miles) between two points using geoDistance([a,b]) or compute the length of a GeoJSON Polygon or Linestring using geoLength(object). You can also find the point at a given position along the great arc between two points using geoInterpolate([a,b]), which returns a function that takes a parameter between 0 and 1, and returns the coordinates of the point at that position along the great arc. For example geoInterpolate([a,b])(0.5) gets the midpoint. To get the pixel distance, use geoPath.measure(object), which returns the projection-dependent length of the path in pixels.

This example measures the length of the route, and prints (console) the coordinates of the midpoint, in geo coords and in pixel coords. The tooltip, placed at the midpoint, shows the distance in pixels (depends on the projection) and km and miles (which don't depend on the projection). Try different projections.
