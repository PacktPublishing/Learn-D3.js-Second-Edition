Projections in D3

These pages plot maps with all projections available in the d3-geo, d3-geo-projection and d3-geo-polygon modules from D3 7.9

  • Main projections. Popular azimuthal, conic and cylindrical projections.
  • Special projections focused on parts of the globe.
  • Selected projections. A selection of popular projections.
  • All d3-geo-projection projections.
  • All d3-geo-polygon projections.

See JavaScript code in the js folder and CSS in the css folder.
