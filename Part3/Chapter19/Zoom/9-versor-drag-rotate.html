Versor drag rotation implemented with d3.drag

This implementation uses a quaternion (instead of euler angles) to compute rotation while dragging. This allows greater control and a correct response. The implementation is based on the versor library by Jason Davies and Mike Bostock, which provides functions to convert between Euler angles and quaternions, compute the difference between two quaternions, and apply a quaternion rotation to another quaternion. See the GitHub repository for versor.js, a Tutorial about versor dragging by Jason Davies and an interactive notebook by Mike Bostock.

This example doesn't implement zooming. See the next example for a version that implements both dragging and zooming using d3.zoom.
