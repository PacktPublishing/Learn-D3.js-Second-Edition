Rotation implemented with d3.geoInertiaDrag

Drag the mouse to rotate the globe with an inertial effect. Zooming is supported using the mouse wheel only.

This is a quaternion-based inertial implementation using the d3.geoInertiaDrag function from the d3-inertia library, by Philippe Rivière. This library is based on the versor library and very easy to use. It is available from this GitHub repository.
