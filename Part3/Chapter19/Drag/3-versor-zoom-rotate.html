Versor drag rotation implemented with d3.zoom

Drag the mouse to rotate the globe. Zooming is supported using the mouse wheel or double-click.

This implementation uses a quaternion (instead of euler angles) to compute rotation while dragging. This allows greater control and a correct response. The implementation is based on the versor library by Jason Davies and Mike Bostock, which provides functions to convert between Euler angles and quaternions, compute the difference between two quaternions, and apply a quaternion rotation to another quaternion. See the GitHub repository for versor.js, a Tutorial about versor dragging by Jason Davies and an interactive notebook by Mike Bostock. See also an example with d3.drag that does not support zooming.
