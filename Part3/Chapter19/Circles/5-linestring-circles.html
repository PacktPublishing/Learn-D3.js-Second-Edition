Drawing resampled GeoJSON LineStrings as circles

This map draws circles using resampled GeoJSON LineString geometries (resampling uses interpolation to add intermediate points so that the line can be adjusted to the projection). See js/wgs84_resampling.js. The file 3-hemispheres-shadows.html obtains the same effect using d3.geoCircle() (see drawCircles() in js/hemispheres.js.
