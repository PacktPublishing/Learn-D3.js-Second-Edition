Using TopoJSON

TopoJSON is a more compact and efficient format for spatial data. It removes redundant coordinates by encoding shared boundaries between features as arcs and applies quantization that reduces file size. TopoJSON also makes it easier to compute derived geometries meshes, merged features and neighbors. To use TopoJSON in D3 you need to convert its data to GeoJSON using the topojson-client library.

This file is identical to 11-multi-theme-choropleth.html, but uses a TopoJSON file instead of GeoJSON to render the features. The TopoJSON data file is world-lowres.topojson. It was generated from the GeoJSON file using online services.
