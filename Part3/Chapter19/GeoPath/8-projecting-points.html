Placing points using the projection function

The projection function works like a scale converting spatial coordinates to screen coordinates. Use it to place points on the map knowing their latitudes and longitudes. This example projects cities on the viewport rendered as SVG <circle> elements.

This solution works well with projections that show the entire globe. Using an orthographic projection (click on the map to switch) will not hide cities that are on the hidden side (e.g. Tokyo, Sydney, Beijing). The simplest solution is to use GeoJSON 'Point' geometries (see next example).
