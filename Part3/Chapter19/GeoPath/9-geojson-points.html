Placing points using GeoJSON

Points can also be provided as a GeoJSON file, and if they are not, you can add them to a GeoJSON object to render them using d3.geoPath. This file is identical to 8-projecting-points.html, but places the coordinates in Point GeoJSON geometries. This time, when you switch to the orthographic projection, points that are on the hidden side of the globe will not be rendered, since they are clipped. Adding the text labels, however, requires translating to the projected centroid of the point geometry, using geoPath.centroid.
