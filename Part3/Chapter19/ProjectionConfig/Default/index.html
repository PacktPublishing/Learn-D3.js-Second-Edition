Basic map used to demonstrate projection configuration (Mercator)

See different configurations in the files in the subfolders:

  1. Fit: fitting the projection in the viewport (pixels).
  2. Center: setting the geographic center (longitude and latitude).
  3. Translate, scale and angle: transforming the projected map (pixels).
  4. Clip: clipping the projection.
  5. Rotate: rotating the globe using geographic coordinates (degrees).
  6. Invert: getting geo coordinates from pixel positions.
