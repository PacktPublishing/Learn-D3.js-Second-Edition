Clipping a map

Three ways to clip a map: using geographic coordinates (clipAngle), using pixel coordinates (clipExtent) or using SVG clipPath. Each technique applies to different purposes.

  • Geo coord clipping - Orthographic. Clipping by angle radius.
  • Geo coord clipping. Orthographic with reference underlay.
  • Geo coord clipping - Mercator. Clipping by angle radius.
  • SVG clipping. For interrupted projections in d3-geo-projection.
  • Screen clipping - Orthographic. Using pixel coordinates.
  • Screen clipping - Mercator
