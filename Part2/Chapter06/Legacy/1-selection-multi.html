Example using d3-selection-multi (deprecated)

The d3-selection-multi module allows the use of a single method to apply multiple styles and attributes. Although convenient, it is currently deprecated and not recommended. Itwas very popular in older versions of D3 and still works, but it is not bundled with d3.js and conflicts with ESM usage of D3. If you have legacy code that uses it, either use with npm (with a JavaScript Node dev environment), or load the scripts (not using ESM) as in this file.
