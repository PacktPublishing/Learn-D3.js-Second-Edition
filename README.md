# Learn-D3js-2025
Repository for the second edition of Learn D3.js, a comprehensive guide to mastering D3.js for data visualization. This repository contains all the code examples, exercises, and resources used in the book.

## 1) Description of each subfolder in this folder

__`Part1/`__

Contains code examples and exercises related to the first part of the book, which covers the fundamentals of D3.js, including data binding, scales, axes, and basic visualizations.

__`Part2/`__

Contains code examples and exercises related to the second part of the book, which focuses on core D3.js techniques, such as selections, data joins, axes, scales, data management, layout managers, basic charts, and more advanced core topics such as transitions, animations, and interactivity.

__`Part3/`__

Contains code examples and exercises related to the third part of the book, which explores advanced D3.js techniques and complex data visualizations, including hierarchical layouts, network and geographic visualizations.

__`Extra/`__

Contains additional code examples, exercises, and resources that complement the content covered in the book, including bonus chapters (tutorials), integration with frontend frameworks, tools for processing datasets, and other topics.
